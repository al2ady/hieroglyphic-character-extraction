# Hieroglyphic Character Extraction using Image Processing

This project focuses on extracting hieroglyphic characters from images using classical image processing techniques implemented in Python.  
It demonstrates a complete preprocessing pipeline for segmenting and isolating characters from complex backgrounds.

The project follows these main steps:
